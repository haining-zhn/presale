# presale商机管理系统
商机增删改查、图表统计管理等。
## 关于作者：https://ytyzhn.top/about.html
## 1、本地环境搭建
建议使用集成环境phpstudy2018，网址：https://www.xp.cn/
版本选择：php5.6 + apache 设置好MySQL密码。
## 2、数据库配置
创建数据库presale，导入presal.sql文件，前端修改PC->Config-config <br />
	'DB_TYPE'					=>	'mysql',  <br />
	'DB_HOST'					=>	'localhost',<br />
	'DB_NAME'					=>	'presale',<br />
	'DB_USER'					=>	'root',<br />
	'DB_PWD'					=>	'root密码',<br />
	'DB_PORT'					=>	'3306',<br />
	'DB_PREFIX'					=>	'tb_',<br />
## 3、登录系统
